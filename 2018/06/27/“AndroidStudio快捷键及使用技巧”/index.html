AndroidStudio快捷键及使用技巧

Android 小技巧 2018/06/27

本文于741天之前发表，文中内容可能已经过时。

  • 自动导包，自动删除没有调用的包：打开AndroidStudio界面之后：依次点击 File –> setting –> Editor –> General –>Auto import
  • 当前类/方法被引用的位置查找：
  1. 先光标定位到方法或者类
  2. 右键选择Find Usages（快捷键 mac:fn+option+F7 win:ctrl+g）
  • 全局搜索 Edit->find->find in path 快捷键 shift+command+Fd
  • 查找一个文件的引用fn+option+F7
